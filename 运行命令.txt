# 生物化学研究资讯抓取与推送系统 - 运行命令

## 基本运行命令

### Windows 系统
python -m app run

### 查看帮助
python -m app --help

### 测试数据源
python -m app test-sources

### 测试特定数据源
python -m app test-sources --source biorxiv
python -m app test-sources --source pubmed
python -m app test-sources --source rss

## 运行参数

### 指定抓取窗口天数（默认1天）
python -m app run --window-days 2

### 指定Top K论文数量（默认12篇）
python -m app run --top-k 20

## 定时任务设置

### Windows 定时任务
使用 run_daily.bat 文件，或通过任务计划程序设置

### 快速设置定时任务
参考：快速设置定时任务.txt

## 注意事项

1. 确保已安装所有依赖：pip install -r requirements.txt
2. 确保已配置 API 密钥（在环境变量或 .env 文件中）
3. 数据库文件位置：data/paper_push.db
4. 报告文件位置：reports/生化领域突破汇总_YYYYMMDD.txt
5. 日志文件位置：logs/paper_push_YYYYMMDD_HHMMSS.log

## 常见问题

### 如果遇到 API 连接错误
- 检查网络连接
- 检查 API 密钥是否有效
- 查看日志文件了解详细错误信息

### 如果遇到数据库错误
- 检查 data/ 目录是否存在
- 检查数据库文件权限

### 如果遇到依赖错误
- 运行：pip install -r requirements.txt
- 确保 Python 版本 >= 3.8

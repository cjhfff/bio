# 不相关论文筛选问题分析

## 问题根源

### 当前筛选流程的问题

1. **数据源抓取阶段**（过于宽松）
   - 只要论文标题/摘要包含关键词（如"structure"、"enzyme"、"signal"）就被抓取
   - 例如：`"Crystal structure of cancer biomarker"` 会被抓取，因为包含"structure"

2. **评分阶段**（只看关键词，不看上下文）
   - 评分系统只看关键词是否存在，不看关键词的上下文
   - 例如：
     - ✅ 相关：`"Enzyme structure and mechanism"` → 高分
     - ❌ 不相关：`"Crystal structure of cancer biomarker"` → 也高分（因为包含"structure"）

3. **没有语义理解**
   - 系统不知道论文的实际研究方向
   - 只看表面关键词匹配

### 具体问题示例

从日志可以看到，以下论文被评分系统选中，但AI判断为不相关：

1. **论文1**: "Structural basis for non-AUG translation regulation"
   - 评分：130分（包含"structural"关键词）
   - 实际：翻译调控研究，不属于三大方向

2. **论文3**: "A Generative Foundation Model for Cryo-EM Densities"
   - 评分：127分（包含"cryo-em"关键词）
   - 实际：AI/机器学习工具，不属于三大方向

3. **论文4**: "Large scale prospective evaluation of co-folding"
   - 评分：124分（包含结构相关关键词）
   - 实际：蛋白质折叠预测工具，不属于三大方向

## 解决方案

### 方案1：改进评分系统（推荐）

在评分时要求关键词必须在相关上下文中：

```python
# 当前逻辑（有问题）
if "structure" in text:
    score += 20  # 任何包含"structure"的都给分

# 改进逻辑
if "structure" in text:
    # 检查是否在相关上下文中
    if any(context in text for context in ["enzyme structure", "protein structure", "molecular structure"]):
        score += 20  # 相关上下文，给分
    elif any(context in text for context in ["crystal structure of", "structure of cancer", "structure of disease"]):
        score += 0  # 不相关上下文，不给分
```

### 方案2：在评分后快速AI预筛选（更准确但更慢）

在评分后、详细分析前，先用AI快速判断是否相关：

```python
# 对高分论文（>50分）先用AI快速判断
if scored_paper.score > 50:
    is_relevant = quick_ai_check(scored_paper.paper)  # 快速AI判断
    if not is_relevant:
        scored_paper.score = 0  # 不相关，分数归零
```

### 方案3：改进关键词匹配规则（平衡方案）

要求论文必须同时满足：
1. 包含关键词
2. 不包含明显的排除领域词（如"cancer"、"disease"、"clinical"等，除非是结构生物学研究）

## 建议实施

**短期方案**（立即实施）：
- 在评分系统中添加"上下文检查"
- 如果关键词出现在不相关上下文中，降低分数或直接过滤

**长期方案**（需要测试）：
- 对高分论文（>50分）先用AI快速判断是否相关
- 如果不相关，降低分数或直接过滤
